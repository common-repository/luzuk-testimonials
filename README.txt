Testimonial Post and Slider is a plugin to create your own testimonial section.

== Description ==

Luzuk Testimonial plugin is the one of the important or most attractive plugin with the help of this we can easily create our own Testimonial in that we show the client reviews or feedback to our own WordPress website.

The Testimonial Plugin are easy to use in any WordPress themes. Here we could add or create the Testimonial Manually by copying there short codes into your post or pages.

In that we can use the various layout , slider , Animation , Gried , Colors or We can set the Images there Height,Width or Font size as per our need. with this all options our testimonial looks more Attractively and Work fastly without taking a Time .

Our Shortcodes

* [Luzuk_Testimonial]
* [Testimonial_Slider]

== Installation ==

= Minimum Requirements =

* PHP version 5.4 or greater
* MySQL version 5.0 or greater

= We recommend your host supports: =

* PHP version 7.0 or greater
* MySQL version 5.6 or greater
* WordPress Memory limit of 64 MB or greater (128 MB or higher is preferred)

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
